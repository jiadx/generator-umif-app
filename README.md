# generator-umif-app

##### Install required tools `yo`:
```
npm install -g yo
```

##### Install `generator-umif-app`:
```
npm install -g generator-umif-app
```
### Run

##### Create a new directory, and go into:
```
mkdir Demo && cd $_
```

##### Run `yo umif-app`, and select desired configurations:
```
yo umif-app
```
